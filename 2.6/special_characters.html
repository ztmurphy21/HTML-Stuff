Sample Description List with Special Characters

TCP
Transmission Control Protocol is a method used along with the Internet Protocol tro send data in the form of message units between computers over the Internet.
IP
Internet Protocol is the method by which data is sent from one computer to another on the Internet. Each computer on the Internet is uniquely identified by each IP Address.
FTP
File Transafer Protocol is a protocol used to exchange files between computers on the Internet.
HTTP
Hypertext Transafer Protocol is a method used for echanging text, graphics, sound, and other multimedia over the web.
